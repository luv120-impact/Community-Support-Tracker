1. Purpose of This Style Guide

This document defines the shared design rules for all components (Donation Tracker, Volunteer Hours Tracker, Event Signup).
It ensures that every page looks consistent, clean, and professional, regardless of which student developed it.

2. Color Theme
Purpose	Color	Hex
Primary (header, navigation, buttons)	Blue	#2A6DB0
Secondary (background sections)	Light Gray	#F2F2F2
Accent (highlights, hover states)	Amber	#FFB300
Text – Main	Dark Gray	#222222
Text – Secondary	Medium Gray	#555555
Text on Blue Areas	White	#FFFFFF
3. Typography

Font Family (used site-wide):

font-family: Arial, Helvetica, sans-serif;


Headings (h1–h3):

Weight: 600–700

Color: #222

Line height: 1.3

Consistent spacing:
margin-top: 1.25rem; margin-bottom: 0.75rem;

Body text:

Size: 16px

Line height: 1.5

Color: #222

4. Layout Rules

Page Structure (applies to ALL pages):

Shared header, navigation bar, and footer

Main content wrapped in:

<main style="max-width: 900px; margin: 0 auto; padding: 20px;">


Spacing:

16px spacing between form fields

24px spacing between main page sections

Full responsive layout (mobile first)

5. Buttons

Default Style:

background: #2A6DB0;
color: #fff;
padding: 10px 16px;
border: none;
border-radius: 6px;
font-weight: 600;
cursor: pointer;


Hover Style:

background: #195487;

6. Form Design

All forms must follow the same structure for consistent UX.

Form rules:

Labels appear above inputs

Required fields marked with an asterisk (*)

Input styling:

padding: 10px;
border: 1px solid #ccc;
border-radius: 4px;
width: 100%;


Error messages:

color: red;
font-size: 14px;
margin-top: 6px;

7. Table Style (Used in Stage Two)

Header Row:

background: #2A6DB0;
color: #fff;
font-weight: 600;


Row Styling:

Row padding: 12px

Alternate row color:

background: #F7F7F7;


Borders:
Thin, light gray border: 1px solid #ddd

8. Mobile Responsiveness

All pages must be readable and usable at 375px width.

Requirements:

Navigation converts to vertical stacked links on small screens

Buttons expand to 100% width

Forms expand to 100%

Avoid horizontal scrolling
